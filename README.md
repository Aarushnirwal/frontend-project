SkillSwap – Student Skill Exchange Platform

SkillSwap is a responsive frontend web application built using HTML, CSS, and JavaScript. The platform helps students connect, share skills, and learn from each other through a simple and interactive interface.

Features
Responsive design
Modern user interface
Interactive skill cards
Skill exchange form
Mobile-friendly layout
Technologies Used
HTML5
CSS3
JavaScript
Purpose

The project is designed to encourage collaborative learning and skill sharing among students.

Future Enhancements
Login & Signup system
Real-time chat
Database integration
User profiles
Search and filter functionality
